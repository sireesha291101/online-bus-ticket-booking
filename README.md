# Online Bus Ticket Booking System

##  Project Overview

The Online Bus Ticket Booking System is a beginner-friendly web application built using HTML, CSS, and JavaScript. It enables users to book bus tickets by entering passenger details, selecting a destination, and specifying the number of tickets. The application calculates the total fare and displays a booking confirmation.

##  Features

- Passenger information form
- Destination selection
- Ticket fare calculation
- Ticket quantity validation
- Age-based discount support
- Booking summary display
- Simple and responsive user interface

##  Technologies Used

- HTML5
- CSS3
- JavaScript
